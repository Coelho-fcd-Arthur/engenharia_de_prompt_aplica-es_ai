# 🤖 Engenharia de Prompt e Aplicações de IA

Este repositório foi criado para a disciplina **Engenharia de Prompt e Aplicações de Inteligência Artificial** da faculdade.
O objetivo é explorar técnicas de criação de prompts eficientes para modelos de IA, além de desenvolver aplicações práticas utilizando ferramentas e APIs de Inteligência Artificial.

---

## 📚 Objetivos do Projeto

* Estudar os fundamentos de **Engenharia de Prompt**
* Aprender a interagir de forma eficiente com **modelos de IA generativa**
* Desenvolver aplicações que utilizem **IA para resolver problemas reais**
* Documentar experimentos, exemplos de prompts e resultados

---

## 🧠 Conteúdos Abordados

* Conceitos básicos de Inteligência Artificial
* Fundamentos de Engenharia de Prompt
* Estrutura e otimização de prompts
* Técnicas como:

  * Zero-shot prompting
  * Few-shot prompting
  * Chain-of-thought prompting
* Integração com APIs de IA
* Desenvolvimento de aplicações baseadas em IA
